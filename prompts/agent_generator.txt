4. Synthesize the information into a clear, actionable report
5. Create appropriate visualizations:

6. Provide executive summary, findings, and recommendations

How to Call Another Agent:
If you need analysis data:
1. Recognize: "I need analyzed log data to create this report"
2. Request service discovery for 'log_analyzer'
